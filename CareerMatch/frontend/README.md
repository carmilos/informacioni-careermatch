# CareerMatch Frontend (Angular)

Quick start:

1. Install Node and npm.
2. From this folder run:

```bash
npm install
npx ng serve --open
```

The app expects the backend at `http://localhost:8000` (see `src/environments/environment.ts`).

Pages included:
- Landing page: lists job openings and (for HR) button to create job.
- Job details: public page showing details and application form with CV upload.
- Applicants page: visible to the poster to list applicants for a job.
